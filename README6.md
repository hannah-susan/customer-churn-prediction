# 🔄 Customer Churn Prediction System

A Machine Learning web app that predicts whether a customer will **Churn** or **Stay**!

## 🚀 Live Demo
[Click here to view the app](https://customer-churn-keerthi.streamlit.app)

## 📌 Project Overview
Customer churn is a major challenge for businesses, as losing customers directly impacts revenue and growth. This project provides an automated, data-driven approach to identify customers at risk of churning based on their behavioral and demographic attributes.

## 📊 Dataset
- **Dataset:** Customer Churn Prediction Dataset
- **Features:** Age, Gender, Income, Spending Score, Purchase Amount, Product Category, Payment Method, and more
- **Target:** Churn (Yes/No)

## 🤖 Model Used
- Random Forest Classifier

## ⚙️ Techniques Applied
- Label Encoding
- Standard Scaling
- Train-Test Split (80/20)
- Missing Value Handling

## 📈 Metrics Evaluated
- Accuracy Score

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Joblib
- Streamlit

## 🔑 Key Insights
- Low spending score → higher churn risk
- Inactive customers → higher churn risk
- High returns → higher churn risk
- Low review score → higher churn risk
